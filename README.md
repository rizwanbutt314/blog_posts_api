## Blog Posts API

The purpose of this api is to get the posts data from hatchways api and serve
under /api/posts endpoint.

###Pre-reqs:
* Python: 3.8+
* OS: Linux

### Steps to run app:
* `pip install -r requirements.txt`
* `env FLASK_APP=manage.py flask run`

### Steps to run tests:
* `env FLASK_APP=manage.py flask test`
